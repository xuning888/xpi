# @xuning/xpi

Pi Extension: Sub-agent system with multi-agent coordination, team management, and task tracking

## Installation

```bash
npm install @xuning/xpi
```

## Features

- Multi-agent coordination
- Team management
- Task tracking
- Built-in agents: general-purpose, Explore, Plan

## Usage

This is a Pi extension. Install it in your Pi environment to enable sub-agent capabilities.
